SARA Airlines

If you want to become a good UX/UI designer: Have Idea and Be Focus.


Sara Airlines is considered as one of the airlines which provides the cheapest tickets to its customers. The prices vary from seat to seat. It depends on which seat you are choosing Economic, Premiere, or First class. This app can use easily by anyone either he is a teenage, young, or older person.

No doubt, there are a lot of apps like this which are available in the market. But, these apps don’t provide the exact services which they had shown to the people even in the high rates. Other apps usually have a high price with fewer facilities like food, discounts, return money in a certain period of time and so on. Whenever we compare prices for other apps, generally, they have different but high prices.

Just because of these differences, I decided to make a SARA Airlines app which provides you various facilities like you can easily choose any seat you want from the different class. You can compare the prices of each other. Moreover, it is very easy to pay an amount of the ticket.

As this project is only based on the concept of both UX/UI. So, it became indispensable to use all the design principles in this project. It is tested by the real clients to observe the navigation and design principles.

User Interface

• As the UI is based on the visual discipline. Especially, an apps’ designing should be related to its subject, customers, and users.

• The colors in this app were most challenging because this app is for all the peeps i.e. from younger to older.

User Experience

• The idea behind this app was simple and efficient to use. UX is known for emotionally attached to the users.

• As its an app, so it is important that the visitor should not be confused while using an app. It should be “Get, Set, Go”.

• Colors, Negative space, and Z-Index are done with both the concepts of UX and UI.

The software makes this app responsive, easily. So, one can use this app in the following screens:

  • Web
  • Tablet
  • Mobile
